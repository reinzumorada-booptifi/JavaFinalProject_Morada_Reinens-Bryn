# Student Grade Analyzer System 

## Problem Statement 
This program solves the problem of analyzing student grades efficiently. Instead of manually computing averages, 
highest and lowest scores, and grade performance, this Java console application automatically processes numerical 
grades and provides accurate statistical analysis and academic performance evaluation.

## Program Features 
-Accepts multiple student grades from the user
-Calculates average, highest, lowest, median, and range
-Converts numerical grades into letter grades (Aâ€“F)
-Displays grade distribution and percentages
-Provides performance remarks and recommendations
-Uses object-oriented programming concepts
